# 🌾 Smart Crop Advisory System

AI-powered decision support system for small and marginal farmers.

## Features
- 🌾 **Crop Recommender** — Suggests best crops based on soil NPK, pH, and climate
- 🌿 **Disease Detector** — Upload a leaf photo to detect disease instantly

## Models
- Random Forest Classifier — 99.2% accuracy across 22 crops
- MobileNetV2 Transfer Learning — 96%+ accuracy across 38 disease classes

## Tech Stack
Python · Scikit-learn · TensorFlow · Streamlit · GitHub
